tags.pub

This is a server for hashtags on the ActivityPub network.

It complies with ActivityPub's requirement that object IDs are https: URLs that resolve to an Activity Streams 2.0 representation.

Using tags.pub

You can just refer to a tags.pub URL when using hashtags in your code. For example, 

              {
                "@context": "https://www.w3.org/ns/activitystreams",
                "type": "Note",
                "content": "Here is a note with a tag.",
                "tag": "https://tags.pub/tag/example"
              }
or 
              {
                "@context": "https://www.w3.org/ns/activitystreams",
                "type": "Note",
                "content": "Here is a note with a tag.",
                "tag": {
                  "id": "https://tags.pub/tag/example",
                  "type": "Hashtag"
                }
              }

If you fetch the object at https://tags.pub/tag/example, you will get something like the following: 

              {
                "id": "https://tags.pub/tag/example",
                "type": "Hashtag",
                "name": "#example"
              }
